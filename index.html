Perfil De Egreso

La formacion que ofrece la carrera de Tecnico en programacion permite al egresado, a travez de la aricularizacion de saberes de diversos campos, realizar actividades dirigidas a la:

  • Instalaciones y desrrollo de software de aplicacion utilizando programacion estructurada y orientada a objetos en ambientes web y moviles, con almacenamiento persistentes de datos,
  • Asi como la configuracion y administracion de plataforma e-learning y
  • Comercio electronico
Durante el proceso de formacion de los cinco modulos, el estudiante desrrolla o reforzara:
Las siguientes competencias profecionales:
  • Desarrolla e instala software de aplicacion utilizando programacion estructurada, con almesenamiento persistente de los datos
  • Desarrolla software de aplicacion utilizando utilizando programacion orientada a objetos, con almacenamiento percistente de los datos
  • Desrrolla aplicaciones web y moviles
  • Adnistra sistemas operativos, aplicaciones y servicios
  • Desarrolla, admistra y configura soluciones de e - learnig y comercio electronico

  • Modulo I

    Desarrollo e intalacion de software de aplicaciones utilizando programacion estructurada, con almacenamiento persistente de los datos

  • Modulo II

    Desarrolla software de aplicacion utilizando programacion orientada a objetos con almacenamiento persistente de los datos

  • Modulo III

    Desarrolla aplicaciones web y moviles

  • Modulo IV

    Admintra sistemas operativos, aplicaciones y servicios

  • Modulo V

    Desarrolla, admistra y configura soluciones de e - learning y comercio electronico
